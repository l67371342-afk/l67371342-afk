# infADVOKAT

**infADVOKAT** — bu onlayn huquqiy maslahat va advokatlik xizmatlarini taklif qiluvchi web-platforma. Loyihamiz foydalanuvchilarga tezkor, ishonchli va intuitiv interfeys orqali huquqiy maslahat olish imkonini beradi.

---

## 🔹 Xususiyatlar
- Foydalanuvchi ro‘yxatdan o‘tishi va profil yaratishi mumkin
- Huquqiy maslahatlar bo‘yicha interaktiv forum
- Admin panel orqali ma’lumotlarni boshqarish
- Mobil va desktop uchun moslashuvchan dizayn

---

## 🔹 Texnologiyalar
- HTML, CSS, JavaScript (yoki React / Next.js)
- GitHub & Vercel orqali deploy
- Optional: Backend uchun Node.js yoki Firebase
